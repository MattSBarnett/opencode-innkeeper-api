# Innkeeper Hotel Booking API

## Overview
This is the API service for the Innkeeper Hotel Booking System. It handles all database interactions and exposes REST endpoints for booking management.

## Tech Stack
- Java 17
- Spring Boot 3.x
- Spring Data JPA
- PostgreSQL
- Maven

## Running the API
1. Ensure PostgreSQL is running.
2. Configure database credentials in `src/main/resources/application.properties`.
3. Run the application using Maven:
   ```bash
   mvn spring-boot:run
   ```
The API runs on port 8081.
